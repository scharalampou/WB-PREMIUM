# **App Name**: WinBloom

## Core Features:

- Daily Win Input: Text input for the user to log their daily win.
- Gratitude Input: Text input for the user to log what they are grateful for.
- Dewdrop Counter: Increments by 10 for each daily log. LocalStorage maintains count on refresh. Save counter progress locally using LocalStorage. Show a fun and visually pleasing number
- Blooming Garden: For every 50 Dewdrops earned, a new Lottie-animated flower blooms, visually representing user progress and persistance.
- Win Suggestion Tool: A shuffle button generates witty/funny suggestions to inspire win logging; this tool can overcome user blocks about small accomplishments.
- Global Feed: Displays 20 pre-written anonymous example wins to showcase the vibe of the community.
- LocalStorage Persistence: Saves Dewdrops and Wins locally, ensuring garden progress and log history persist across refresh. Use LocalStorage

## Style Guidelines:

- Primary color: Deep Emerald (#064E3B) to evoke a sense of growth and tranquility.
- Background color: Desaturated Emerald (#F0FDF4) provides a soft, supportive backdrop to ensure comfortable reading and viewing of the WinBloom app's information.
- Accent color: Marigold (#D97706) is employed sparingly as an accent for key interactive elements, ensuring affordance is obvious on the user interface.
- Headline font: 'Playfair', serif, with a geometric, high-contrast style, is used for headlines and small amounts of text, to add elegance. Body font: 'PT Sans' sans-serif, a humanist font which complements Playfair, ensures the content's body is comfortable to read.
- Use nature-themed icons related to growth and positivity.
- Central seedling as the main focus, with wins and gratitude input fields surrounding it.
- Lottie-animated flower blooming on milestone achievements, celebrating growth and consistency.
